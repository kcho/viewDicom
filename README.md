# viewDicom

```

usage: viewDicom.py [-h] [-i DICOMDIR] [-s SLICENUMBER] [-o OUT] [-v]

viewDicom.py : Visualize the dicoms using matplotlib
========================================

optional arguments:
  -h, --help            show this help message and exit
  -i DICOMDIR, --dicomDir DICOMDIR
                        Data directory location, default=pwd
  -s SLICENUMBER, --sliceNumber SLICENUMBER
                        Number of horizontal slice
  -o OUT, --out OUT     png output location
  -v, --verbose         Calculate whole image array


```
